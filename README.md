# EnhancifAI Backend

## Overview
EnhancifAI Backend is a Python-based service designed for processing CSV files using AI algorithms. It integrates functionalities for handling CSV files and interacting with OpenAI's API.

## Files
- `csv_handler.py`: Handles CSV file operations.
- `openai_api.py`: Manages interactions with OpenAI's API.
- `run_enhancifai.py`: The main script to run the backend service.
- `server.py`: Sets up a server for handling processing requests.

## Setup
1. **Install Dependencies**: Ensure Python is installed and install necessary packages.
2. **API Keys**: Obtain and set up your OpenAI API keys.
4. **Running the Server**: Use `run_enhancifai.py` to process CSV files with AI.
